# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-15

### Added

-   Initial release of UCheck.
-   `add` command to add a URL to the queue.
-   `list` command to list all URLs in the queue.
-   `run` command to concurrently check the status of all URLs.
-   `clean` command to clear the URL queue.
-   `README.md` with project information and usage instructions.
-   `LICENSE` file (MIT).
